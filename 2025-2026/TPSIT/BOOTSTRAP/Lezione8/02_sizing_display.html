Width — larghezza percentuale

Le classi w-* impostano la larghezza come percentuale del contenitore padre.

w-25   → larghezza 25%
w-50   → larghezza 50%
w-75   → larghezza 75%
w-100  → larghezza 100%
w-auto → larghezza automatica (in base al contenuto)

Attenzione: w-* funziona solo rispetto al contenitore padre. Se il padre non ha una larghezza definita, potrebbe non avere l'effetto atteso.

Height — altezza percentuale

Le classi h-* impostano l'altezza come percentuale del contenitore padre.
Attenzione: il padre deve avere un'altezza definita, altrimenti h-* non funziona.

h-25  → altezza 25% del padre
h-50  → altezza 50% del padre
h-75  → altezza 75% del padre
h-100 → altezza 100% del padre

Display Utilities

Le classi d-* controllano la proprietà CSS display.

d-none         → display: none        (nascosto, non occupa spazio)
d-block        → display: block       (blocco, occupa tutta la larghezza)
d-inline       → display: inline      (inline, si affianca al testo)
d-inline-block → display: inline-block (inline ma accetta larghezza/altezza)
d-flex         → display: flex        (contenitore flessibile)
d-inline-flex  → display: inline-flex (flex ma inline)
d-grid         → display: grid
