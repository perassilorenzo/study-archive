Alerts e Badges

Messaggi di avviso e etichette colorate

Alert base

Combina alert con una classe colore per definire il tipo di messaggio.

<div class="alert alert-success">Operazione riuscita!</div> <div class="alert alert-danger">Si è verificato un errore.</div> <div class="alert alert-warning">Attenzione, controlla i dati.</div> <div class="alert alert-info">Informazione utile.</div> <div class="alert alert-primary">Messaggio primario.</div> <div class="alert alert-secondary">Messaggio secondario.</div>
Operazione riuscita!
Si è verificato un errore.
Attenzione, controlla i dati.
Informazione utile.
Messaggio primario.
Messaggio secondario.

Alert chiudibile (dismissible)

Aggiungendo alert-dismissible fade show e un pulsante btn-close, l'utente può chiudere il messaggio.

<div class="alert alert-warning alert-dismissible fade show" role="alert"> Questo messaggio può essere chiuso. <button type="button" class="btn-close" data-bs-dismiss="alert"></button> </div>
Attenzione! Questo messaggio può essere chiuso.
Errore! Qualcosa è andato storto. Riprova.
Fatto! I dati sono stati salvati correttamente.

Alert generato con JavaScript

È possibile creare alert dinamicamente tramite JavaScript, ad esempio dopo un'azione dell'utente.

<button class="btn btn-danger" onclick="mostraErrore()">Simula errore</button> <button class="btn btn-success" onclick="mostraSuccesso()">Simula successo</button> <div id="zona-alert" class="mt-3"></div> <script> function mostraErrore() { document.getElementById('zona-alert').innerHTML = '<div class="alert alert-danger alert-dismissible fade show">' + '<strong>Errore!</strong> Operazione non riuscita.' + '<button type="button" class="btn-close" data-bs-dismiss="alert"></button>' + '</div>'; } function mostraSuccesso() { document.getElementById('zona-alert').innerHTML = '<div class="alert alert-success alert-dismissible fade show">' + '<strong>Successo!</strong> Operazione completata.' + '<button type="button" class="btn-close" data-bs-dismiss="alert"></button>' + '</div>'; } </script>

Badges — Colori base

La classe badge + bg-* crea etichette colorate.

<span class="badge bg-primary">Primario</span> <span class="badge bg-success">Successo</span> <span class="badge bg-danger">Pericolo</span> <span class="badge bg-warning text-dark">Avviso</span> <span class="badge bg-info text-dark">Info</span> <span class="badge bg-secondary">Secondario</span> <span class="badge bg-dark">Scuro</span>
Primario Successo Pericolo Avviso Info Secondario Scuro

Badge arrotondato (pill)

Usa rounded-pill per ottenere la forma a capsula.

<span class="badge rounded-pill bg-primary">Primario</span> <span class="badge rounded-pill bg-danger">Pericolo</span> <span class="badge rounded-pill bg-success">Successo</span>
Primario Pericolo Successo Avviso Info

Badge nei bottoni e nei titoli

<!-- Badge nel bottone (contatore) --> <button class="btn btn-primary"> Messaggi <span class="badge bg-light text-dark">4</span> </button> <!-- Badge nel titolo --> <h4>Novità <span class="badge bg-danger">Nuovo</span></h4>
