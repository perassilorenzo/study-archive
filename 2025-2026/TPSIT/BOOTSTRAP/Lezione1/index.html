1. Struttura Base Bootstrap

Ogni pagina Bootstrap deve avere:

  • Il tag <meta name="viewport"> nell'<head>
  • Il link al CSS Bootstrap nell'<head>
  • Lo script JS Bootstrap prima di </body>
<!DOCTYPE html> <html lang="it"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>La mia pagina</title> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet"> </head> <body> <!-- Contenuto --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script> </body> </html>
