Bootstrap - Lezione 5

Navbar, posizionamento, dropdown e icone con Font Awesome

Indice della Lezione

1. Navbar base e stili 2. Navbar responsive e dropdown 3. Font Awesome

1. Struttura base della Navbar

Una navbar Bootstrap si costruisce con <nav> e almeno tre classi fondamentali.

<nav class="navbar navbar-expand-lg navbar-dark bg-dark"> <div class="container"> <a class="navbar-brand" href="#">MioSito</a> </div> </nav>
MioSito

2. Stili e colori

Combina navbar-dark o navbar-light con una classe di sfondo.

navbar-dark bg-dark
navbar-dark bg-primary
navbar-dark bg-success
navbar-dark bg-danger
navbar-light bg-light

3. Posizionamento

ClasseEffetto
(nessuna)Normale nel flusso della pagina
fixed-topFissa in cima, si sovrappone al contenuto
fixed-bottomFissa in fondo alla pagina
sticky-topRimane in cima solo dopo aver scrollato fino a lei
Con fixed-top, aggiungi padding-top al body per evitare che il contenuto finisca coperto dalla navbar.

4. Navbar responsive con hamburger

Su schermi piccoli, i link si nascondono e appare il bottone hamburger navbar-toggler.

<nav class="navbar navbar-expand-lg navbar-dark bg-dark"> <div class="container"> <a class="navbar-brand" href="#">MioSito</a> <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#menuNavbar"> <span class="navbar-toggler-icon"></span> </button> <div class="collapse navbar-collapse" id="menuNavbar"> <ul class="navbar-nav ms-auto"> <li class="nav-item"><a class="nav-link" href="#">Home</a></li> <li class="nav-item"><a class="nav-link" href="#">Chi siamo</a></li> <li class="nav-item"><a class="nav-link" href="#">Contatti</a></li> </ul> </div> </div> </nav>

5. Font Awesome — Icone nei link

Aggiungi il CDN nell'<head> e usa il tag <i> con le classi icona.

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.0/css/all.min.css"> <i class="fas fa-home"></i> <i class="fas fa-envelope"></i> <i class="fab fa-github"></i>
