I Bottoni in Bootstrap

La Regola Base dei Bottoni

Ogni bottone Bootstrap richiede sempre due classi minime:

  1. btn — classe base obbligatoria (fornisce padding, bordi, cursor, ecc.)
  2. btn-[variante] — definisce il colore del bottone
<!-- GIUSTO ✅ --> <button class="btn btn-primary">Clicca qui</button> <!-- SBAGLIATO ❌ — manca btn-* --> <button class="btn">Bottone senza colore</button> <!-- SBAGLIATO ❌ — manca btn base --> <button class="btn-primary">Non funziona</button>

Confronto

← senza stile visibile
Tag HTML: I bottoni si possono creare con <button>, <a> o <input type="button"> — Bootstrap si adatta a tutti.

Stili Base: btn-*

<button class="btn btn-primary">Primary</button> <button class="btn btn-secondary">Secondary</button> <button class="btn btn-success">Success</button> <button class="btn btn-danger">Danger</button> <button class="btn btn-warning">Warning</button> <button class="btn btn-info">Info</button> <button class="btn btn-light">Light</button> <button class="btn btn-dark">Dark</button>

Dimensioni: btn-lg e btn-sm

Si aggiunge una terza classe per modificare la dimensione del bottone.

<button class="btn btn-primary btn-lg">Bottone Grande</button> <button class="btn btn-primary">Bottone Normale</button> <button class="btn btn-primary btn-sm">Bottone Piccolo</button>

Risultato

Outline Buttons: btn-outline-*

I bottoni outline hanno solo il bordo colorato e lo sfondo trasparente. Al passaggio del mouse (hover) si riempiono di colore.

<button class="btn btn-outline-primary">Outline Primary</button> <button class="btn btn-outline-secondary">Outline Secondary</button> <button class="btn btn-outline-success">Outline Success</button> <button class="btn btn-outline-danger">Outline Danger</button> <button class="btn btn-outline-warning">Outline Warning</button> <button class="btn btn-outline-info">Outline Info</button> <button class="btn btn-outline-light">Outline Light</button> <button class="btn btn-outline-dark">Outline Dark</button>

Su sfondo bianco

Su sfondo scuro (btn-outline-light si vede meglio)

Confronto: btn-primary = sfondo pieno blu | btn-outline-primary = solo bordo blu, sfondo trasparente.

Button Groups

Il button group raggruppa più bottoni in una riga unita, eliminando lo spazio tra di loro e unendo i bordi.

Struttura: <div class="btn-group"> che contiene i bottoni.

Button Group Base

<div class="btn-group" role="group"> <button class="btn btn-primary">Sinistra</button> <button class="btn btn-primary">Centro</button> <button class="btn btn-primary">Destra</button> </div>

Button Group con Outline (uso comune: selezione opzioni)

<div class="btn-group" role="group"> <button class="btn btn-outline-primary">HTML</button> <button class="btn btn-outline-primary">CSS</button> <button class="btn btn-outline-primary">JavaScript</button> <button class="btn btn-outline-primary">Bootstrap</button> </div>

Button Group con stili misti

<div class="btn-group" role="group"> <button class="btn btn-success">✅ Approva</button> <button class="btn btn-danger">❌ Rifiuta</button> <button class="btn btn-secondary">⏸ In attesa</button> </div>

Button Group Verticale: btn-group-vertical

<div class="btn-group-vertical" role="group"> <button class="btn btn-outline-success">Opzione 1</button> <button class="btn btn-outline-success">Opzione 2</button> <button class="btn btn-outline-success">Opzione 3</button> </div>

Più Button Group affiancati (Toolbar)

<div class="btn-toolbar" role="toolbar"> <div class="btn-group me-2" role="group"> <button class="btn btn-secondary">1</button> <button class="btn btn-secondary">2</button> <button class="btn btn-secondary">3</button> </div> <div class="btn-group me-2" role="group"> <button class="btn btn-secondary">4</button> <button class="btn btn-secondary">5</button> </div> <div class="btn-group" role="group"> <button class="btn btn-secondary">6</button> </div> </div>

Bottoni come Link

Si può usare il tag <a> con le classi bottone per creare link che sembrano bottoni:

<a href="index.html" class="btn btn-primary">Vai alla Home</a> <a href="02_testo_colori.html" class="btn btn-outline-success">Lezione precedente</a>

Esercizio Pratico

Crea la seguente interfaccia:

  1. Un bottone grande verde con testo "Salva"
  2. Un bottone normale outline rosso con testo "Elimina"
  3. Un group con 3 bottoni: "Giorno" | "Settimana" | "Mese" (stile outline secondary)
Mostra soluzione
<button class="btn btn-success btn-lg">Salva</button> <button class="btn btn-outline-danger">Elimina</button> <div class="btn-group" role="group"> <button class="btn btn-outline-secondary">Giorno</button> <button class="btn btn-outline-secondary">Settimana</button> <button class="btn btn-outline-secondary">Mese</button> </div>
