Ricerca / Filtro tabella con JavaScript

Un campo di testo filtra le righe in tempo reale controllando il contenuto di ogni riga.

<input type="text" id="cerca" class="form-control mb-3" placeholder="Cerca..."> <table class="table" id="tabella">...</table> <script> document.getElementById('cerca').addEventListener('input', function () { const testo = this.value.toLowerCase(); const righe = document.querySelectorAll('#tabella tbody tr'); righe.forEach(function (riga) { riga.style.display = riga.textContent.toLowerCase().includes(testo) ? '' : 'none'; }); }); </script>
