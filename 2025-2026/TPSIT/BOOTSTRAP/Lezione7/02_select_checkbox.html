Select, Checkbox, Radio e Switch

form-select, form-check, form-switch, optgroup, multiple

Select base

Usa form-select (non form-control) per i menu a tendina.

<select class="form-select"> <option value="">-- Scegli un'opzione --</option> <option value="1">Prima opzione</option> <option value="2">Seconda opzione</option> <option value="3" selected>Terza (preselezionata)</option> </select>

Select con optgroup (gruppi)

<optgroup> raggruppa le opzioni con un'intestazione non selezionabile.

<select class="form-select"> <optgroup label="Frutta"> <option>Mela</option> <option>Pera</option> </optgroup> <optgroup label="Verdura"> <option>Carota</option> <option>Patata</option> </optgroup> </select>

Select multipla

Con l'attributo multiple si possono selezionare più voci tenendo premuto Ctrl (o Cmd su Mac).

<select class="form-select" multiple> <option>HTML</option> <option>CSS</option> <option>JavaScript</option> <option>Bootstrap</option> </select>

Checkbox (scelta multipla)

Ogni checkbox usa form-check come contenitore, form-check-input sull'input e form-check-label sulla label.

<div class="form-check"> <input class="form-check-input" type="checkbox" id="check1"> <label class="form-check-label" for="check1">Opzione 1</label> </div> <div class="form-check"> <input class="form-check-input" type="checkbox" id="check2" checked> <label class="form-check-label" for="check2">Opzione 2 (già selezionata)</label> </div> <div class="form-check"> <input class="form-check-input" type="checkbox" id="check3" disabled> <label class="form-check-label" for="check3">Opzione 3 (disabilitata)</label> </div>

Radio button (scelta singola)

I radio con lo stesso name sono in gruppo: selezionarne uno deseleziona gli altri.

<div class="form-check"> <input class="form-check-input" type="radio" name="colore" id="rosso" value="rosso"> <label class="form-check-label" for="rosso">Rosso</label> </div> <div class="form-check"> <input class="form-check-input" type="radio" name="colore" id="blu" value="blu" checked> <label class="form-check-label" for="blu">Blu</label> </div>

Switch (interruttore)

Aggiungendo form-switch al contenitore, la checkbox diventa un interruttore toggle.

<div class="form-check form-switch"> <input class="form-check-input" type="checkbox" id="sw1"> <label class="form-check-label" for="sw1">Attiva notifiche</label> </div>

Checkbox e Radio in linea

Aggiungendo form-check-inline gli elementi si affiancano orizzontalmente.

<div class="form-check form-check-inline"> <input class="form-check-input" type="radio" name="sesso" id="m"> <label class="form-check-label" for="m">Maschio</label> </div> <div class="form-check form-check-inline"> <input class="form-check-input" type="radio" name="sesso" id="f"> <label class="form-check-label" for="f">Femmina</label> </div>
