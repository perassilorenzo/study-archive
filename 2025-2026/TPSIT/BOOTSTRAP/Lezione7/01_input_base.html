Input base e avanzati

form-control, tipi di input, date, color, range, file

Struttura base di un campo

Ogni campo è composto da un <label> e un <input>, avvolti in un div con mb-3 per la spaziatura.

<div class="mb-3"> <label for="nome" class="form-label">Nome</label> <input type="text" class="form-control" id="nome" placeholder="Inserisci il nome"> </div>
Il valore di for nella label deve essere uguale all'id dell'input: così cliccando la label si attiva il campo.

Tipi di input — testo

L'attributo type definisce il tipo di dato accettato e l'aspetto del campo su mobile.

<input type="text"> <!-- testo libero --> <input type="email"> <!-- email con @ obbligatoria --> <input type="password"> <!-- testo nascosto con pallini --> <input type="number"> <!-- solo numeri --> <input type="tel"> <!-- telefono (tastiera numerica su mobile) --> <input type="url"> <!-- indirizzo web -->

Input avanzati

Bootstrap applica form-control anche a questi tipi speciali.

<input type="date"> <!-- selettore data con calendario --> <input type="time"> <!-- selettore ora --> <input type="color"> <!-- selettore colore --> <input type="range"> <!-- cursore scorrevole --> <input type="file"> <!-- caricamento file -->

Textarea

La textarea è un campo di testo multi-riga. L'attributo rows imposta l'altezza iniziale.

<textarea class="form-control" rows="3" placeholder="Scrivi qui..."></textarea> <textarea class="form-control" rows="6"></textarea> <!-- più alta -->

Dimensioni degli input

È possibile rendere un input più piccolo o più grande con form-control-sm e form-control-lg.

<input class="form-control form-control-sm"> <!-- piccolo --> <input class="form-control"> <!-- default --> <input class="form-control form-control-lg"> <!-- grande -->
