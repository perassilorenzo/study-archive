Input Group e Validazione

input-group, input-group-text, is-valid, is-invalid, form layout

Input Group — testo prima o dopo

L'input group permette di aggiungere testo, icone o bottoni direttamente attaccati a un campo.

<!-- Testo PRIMA del campo --> <div class="input-group"> <span class="input-group-text">@</span> <input type="text" class="form-control" placeholder="Username"> </div> <!-- Testo DOPO il campo --> <div class="input-group"> <input type="number" class="form-control" placeholder="Prezzo"> <span class="input-group-text">€</span> </div> <!-- Testo su entrambi i lati --> <div class="input-group"> <span class="input-group-text">https://</span> <input type="text" class="form-control" placeholder="www.sito"> <span class="input-group-text">.it</span> </div>
@
€
https:// .it

Input Group — con bottoni

Un bottone può sostituire lo span dentro l'input group.

<!-- Bottone dopo il campo --> <div class="input-group"> <input type="text" class="form-control" placeholder="Cerca..."> <button class="btn btn-primary">Cerca</button> </div> <!-- Bottone prima del campo --> <div class="input-group"> <button class="btn btn-outline-secondary">Copia</button> <input type="text" class="form-control" value="testo-da-copiare"> </div>

Validazione — is-valid e is-invalid

Aggiungendo is-valid o is-invalid al campo, Bootstrap colora il bordo di verde o rosso. Abbinando valid-feedback o invalid-feedback, appare anche un messaggio sotto il campo.

<!-- Campo valido --> <input type="email" class="form-control is-valid" value="mario@email.it"> <div class="valid-feedback">Email corretta!</div> <!-- Campo non valido --> <input type="email" class="form-control is-invalid" value="non-è-email"> <div class="invalid-feedback">Inserisci un'email valida.</div>
Nome inserito correttamente.
Inserisci un indirizzo email valido (esempio: mario@email.it).
La password deve essere di almeno 8 caratteri.
Città selezionata.

Validazione dinamica con JavaScript

In un caso reale, le classi is-valid e is-invalid vengono aggiunte tramite JavaScript dopo che l'utente ha compilato il campo.

// Aggiunge la validazione al click su Invia document.getElementById('formVal').addEventListener('submit', function(e) { e.preventDefault(); // blocca l'invio reale const email = document.getElementById('emailVal'); if (email.value.includes('@')) { email.classList.remove('is-invalid'); email.classList.add('is-valid'); } else { email.classList.remove('is-valid'); email.classList.add('is-invalid'); } });
Contiene @, sembra una email!
Manca la @, non è una email valida.

Form orizzontale (label + campo affiancati)

Usa row e col-sm-* per mettere label e campo sulla stessa riga. Usa col-form-label per allineare verticalmente la label.

<div class="row mb-3"> <label class="col-sm-3 col-form-label">Nome</label> <div class="col-sm-9"> <input type="text" class="form-control"> </div> </div>
